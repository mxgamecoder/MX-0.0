🚀 How to Use This API

The API returns JSON with keys like content or imageUrl. You can use it in JS, Node, Python, bots, apps, etc.

📌 10 Code Examples

1. JS Fetch (Browser)

fetch("https://mxapi-lnc.onrender.com/fun/joke?meka=your_api_key_here")
  .then(res => res.json())
  .then(data => alert(data.content));

2. Node.js

const https = require("https");
https.get("https://mxapi-lnc.onrender.com/fun/fact?meka=your_api_key_here", res => {
  let d = "";
  res.on("data", chunk => d += chunk);
  res.on("end", () => console.log(JSON.parse(d).content));
});

3. Curl

curl https://mxgamecoder-klfx.onrender.com/fun/quote?meka=your_api_key_here

4. Python

res = requests.get("https://mxapi-lnc.onrender.com/fun/riddle?meka=your_api_key_here")
print(res.json()["content"])
